# Mamma mia memory (Mmm)

## Nombre del juego
**Mamma mia memory (Mmm)**

## Descripción
La temática del juego está inspirada en un restaurante italiano: **trattoria / pizzería**.

La paleta visual usa los colores de la bandera de Italia (verde, blanco y rojo), y la cuadrícula del fondo al iniciar hace referencia al papel encerado antigrasa que se utiliza en las tablas para servir comida, especialmente pizzas.

La idea nace de una experiencia personal: tengo una pizzería y mi día a día gira en torno a ese ambiente.

## Tecnologías usadas
- HTML5
- CSS3
- JavaScript (Vanilla)

## Instalación y uso
No requiere instalación de dependencias.

1. Descarga o clona este repositorio.
2. Abre la carpeta del proyecto.
3. Ejecuta el juego escribiendo `index.html` en la terminal de comandos

Opcional (recomendado en VS Code):
- usar una extensión como Live Server para ejecutar el proyecto en un servidor local.

## Cómo jugar
1. Ingresa tu nombre de chef.
2. Selecciona una dificultad (4x4, 6x6, 8x8).
3. Presiona **Iniciar Juego**.
4. Encuentra las parejas de ingredientes.
5. Gana cuando todas las cartas estén emparejadas.
